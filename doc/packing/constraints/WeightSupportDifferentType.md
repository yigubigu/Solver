## 调用JSON格式

```json
{
	"Name": "WeightSupportDifferentType",
	"WeightMap": {
		"0": 100,
		"1": 100,
		"2": 200
	},
	"SupportMap": {
		"0": 50,
		"1": 50,
		"2": 100
	}
}
```
* Name: String，约束名称。
* WeightMap：一维Integer数组，物品的重量。
* SupportMap：一维Integer数组，物品上表面支撑重量的上限。
